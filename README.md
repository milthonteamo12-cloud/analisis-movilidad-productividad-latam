# 📊 Análisis de Movilidad Urbana y Productividad Económica en LATAM

## 🎯 Objetivo
Evaluar la relación entre congestión vehicular y productividad económica en las principales ciudades latinoamericanas para identificar dónde podría ser prioritaria la inversión en infraestructura de transporte.

## 🌎 Fuentes de datos
- TomTom Traffic Index
- OECD Cities

## 🛠 Herramientas utilizadas
- Python (pandas, numpy)
- Limpieza y transformación de datos
- Análisis exploratorio (EDA)
- Cálculo de correlación estadística
- Visualización de datos

## 📈 Metodología
1. Limpieza y normalización de datasets.
2. Unificación de bases mediante claves comunes.
3. Análisis exploratorio de variables económicas y de movilidad.
4. Cálculo de correlación entre congestión y PIB per cápita.
5. Identificación de ciudades prioritarias.

## 🔍 Principales hallazgos
- Se identificaron diferencias significativas en niveles de congestión entre ciudades.
- Algunas ciudades muestran correlación moderada entre alta congestión y menor productividad.
- La movilidad urbana puede influir en el desempeño económico local.

## 📌 Conclusión
Los resultados sugieren que la inversión estratégica en infraestructura de transporte podría contribuir a mejorar indicadores de productividad en ciertas ciudades latinoamericanas.
